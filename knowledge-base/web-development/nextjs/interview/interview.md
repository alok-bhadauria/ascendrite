Q1: Explain Hydration Mismatch Error in Next.js.
Hydration mismatch happens when the initial HTML rendered on the server differs from the HTML generated by the client component during initial browser layout calculations (e.g. using `typeof window !== 'undefined'` or rendering dates dynamically).

|⚠️ INTERVIEW TRAP: RSC Database connection leak If you import a DB client directly into an RSC inside a nested component that gets rendered multiple times per page load, make sure the client is instantiated as a singleton (like Prisma global client mapping) to avoid database thread pool exhaustion.|
