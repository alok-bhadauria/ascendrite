## 1. Mission Statement
Ascendrite exists to democratize and accelerate advanced technical mastery. We build a portable, metadata-driven learning ecosystem that empowers individuals to acquire deep conceptual understanding and verified practical engineering capabilities. By combining structured curriculum design with localized AI-assisted coaching, we aim to bridge the gap between academic theory and real-world industrial practice.

---

## 2. Core Values
Our work is governed by three foundational values:
*   **Educational Rigor**: We do not simplify complex topics. We believe that true technical mastery requires addressing difficult mathematics, architecture trade-offs, and systems programming challenges.
*   **Information Portability**: The learning experience should not be locked behind proprietary SaaS database schemas. All curriculum, progress definitions, and conceptual mapping files are represented in open, git-friendly formats.
*   **Decoupled Agency**: The platform architecture maintains a strict boundary between learning content (the what) and application logic (the how). This enables both curriculum designers and software engineers to operate independently.

---

## 3. Developer and AI Contributor Alignment
Every contributor (whether human developer or autonomous AI agent) must prioritize modularity, stability, and maintainability. We view documentation and validation schemas as core components of the product itself. Any new feature, content node, or architecture change must be fully documented and validated before integration.
